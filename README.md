# Venekanta

School project for courses SWD4TA032-3011 and SWD4TA033-3008.

## Requirements

- Use Java for backend
- Backend must contain basic CRUD routes
- Use the provided sqlite file as database
- Use JavaScript for frontend 

## Used techniques

### Backend

- Spring Framework
- Hibernate

### Frontend

- Next.js 12.0.4
- Bootstrap 5.1.3 
- JQuery 3.6.0
